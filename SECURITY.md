## Ground Rules

- Please put the following in the `config.json` and attach it to the client object (`client.config.token`):
  - Bot Prefix(es) 💲⁉️
  - Bot Tokens 🏅 
  - Bot Snowflakes ❄️ 
  - User Snowflakes ❄️ 
  - Role Snowflakes ❄️ 
  - Channel Snowflakes ❄️ 
  - Channel Category Snowflakes ❄️ 
  - Separate API Keys 🔐 
  - Color Hex Code(s) ~ {for the sake of convenience} 🎨
